# llama.cpp daily summary — 2026-05-11

Today's activity was moderately busy with 8 commits touching CUDA tensor parallelism, sampling, the HTTP server, and several fixes. The headline change is a large new NCCL-free AllReduce implementation for 2-GPU tensor-parallel inference.

## CUDA / GPU

- **Internal AllReduce kernel for CUDA tensor parallelism** (#22299, `f3c3e0e`) — Introduces a NCCL-free AllReduce path (`GGML_CUDA_ALLREDUCE=internal`) using a pipelined single-kernel approach. Supports FP32/FP16/BF16 with a chunked fallback for large tensors; benchmarks show it beats NCCL by 1.8–15.6% on 2×RTX 5090. Selectable via `GGML_CUDA_ALLREDUCE` env var or `--reduction-provider` flag in `llama-bench`. Currently limited to 2 GPUs, Volta+, NVIDIA only (stubs for HIP/MUSA). **Notable new feature.**

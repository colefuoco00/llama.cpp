# llama.cpp daily summary — 2026-04-21

Active day with 12 commits spanning GPU backends, tensor-parallel fixes, server-breaking changes, and CPU quantization improvements. Two breaking changes stand out: `/api` endpoint removal and a CLI flag rename.

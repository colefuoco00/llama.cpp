# llama.cpp daily summary — 2026-05-12

A busy day dominated by GPU backend improvements across CUDA, Vulkan, OpenCL, and SYCL, alongside a substantial speculative decoding overhaul. The headline change is parallel drafting support in the server, which also introduces chained speculators — the most architecturally significant commit of the day.

## Speculative Decoding

- **Parallel drafting support** — server and `common/speculative` refactored to support multiple simultaneous draft sequences; introduces chained speculators (multiple `spec_type` values, best-wins selection) and async draft eval; breaking change for any downstream code using `common_params_speculative::type` (now a `vector`) — #22838 `68e7ea3`

## CUDA/GPU

- Fix `im2col` grid Y overflow (OW > 65535) for long conv1d audio (e.g. SEANet at 11 s / 16 kHz hit OW ≈ 176000, crashing with `invalid configuration argument`) — #22944 `8e1f9d0`
- Snake fusion hardening: tighter type checks, reject rank > 2 tensors, mirror Vulkan filter for ADD/SUB/MUL/DIV to F32/F16 only — #22912 `e936660`
- Directly include `cuda/iterator` instead of relying on transient `cub/cub.cuh` export — #22936 `8cef820`

# llama.cpp daily summary — 2026-04-29

Today saw 14 commits land on upstream/master, with broad activity across GPU backends (CUDA, Vulkan, WebGPU, CANN), a large WebUI overhaul bringing server-side tool support, and a notable refactor of speculative-decoding params. No breaking API changes were identified, though the `spec` param refactor renames internal categories and may affect downstream tooling that inspects CLI argument metadata.

## CUDA / GPU

- **Blackwell native NVFP4 support** — adds quantized matmul (MMQ/MMVQ) and dequantization kernels for NVIDIA RTX 50-series (Blackwell) using native FP4 instructions; notable new feature (#22196, `fc2b005`)
- Flash-attention support for head sizes DKQ=320 / DV=256 (Mistral Small 4), restricted to GQA=32 via new MMA-f16 and tile kernel configs (#22286, `7b8443a`)

## Vulkan

- **Coalesce Q4_K / Q5_K scale loads** — restructures scale-field reads to avoid SPIR-V bounds-tracking issues in Mesa/ANV; measured 4–9% prompt-processing speedup on Intel Arc and 6% token-gen speedup on Q5_K_S (#21751, `f9f3365`)
- Add pipeline barrier after `writeTimestamp` to fix potential GPU-sync hazard (#21865, `1982117`)
- Improve SPIR-V header detection using `__has_include`, adds FreeBSD support (#21918, `698d19b`)

## WebGPU

- Fix buffer aliasing bug for `ssm_scan`; refactor aliasing logic into the shader-lib layer (#22456, `98bb579`)

## CANN (Ascend)

- Major update: add 7 new ops (`SET`, `CUMSUM`, `FILL`, `DIAG`, `TRI`, `SOLVE_TRI`, `SOFTPLUS`) and fuse GLU/CROSS_ENTROPY_LOSS with vendor kernels; fix `COUNT_EQUAL` correctness bug and ACL graph-cache type-mismatch (#21204, `c3e08f4`)

## API / Server

- **`spec` param refactor** — reorganises speculative-decoding CLI args into a dedicated `sampling` category; renames internal `sparam` → `sampling`; adapts server tasks and tests (#22397, `14e733e`)
- Fix m-rope position tracking: use `pos_next` instead of `n_tokens` in server context (#22439, `516e8d7`)

## WebUI

- **Server tools** — major feature addition integrating the `/tools` endpoint into the web UI; supports built-in, MCP, and JSON-schema tools with permission flows, agentic steering messages, draft messages, and a redesigned settings layout (#21237, `f42e29f`)

## Conversion

- Add GGUF conversion support for NVIDIA Nemotron Nano 3 Omni (#22481, `5d56eff`)

## Common / Fixes
